绝对路径：(全部路径位置)
absolute  绝对的  C:\Users\12414\Desktop\yan.jpg
判断是否为绝对路径：
os.path.isabs('路径')

获取路径： directory  目录  文件夹
当前文件所在文件夹路径

绝对路径
path = os.path.abspath(__file__)  #当前文件完整路径

path = os.getcwd() #当前文件所在的根文件夹 类似os.path.dirname(__file__)
